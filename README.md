# Support Tickets Management System

A full-stack support ticket management application built with Next.js 16 and TypeScript. 

### Core Features
- **Complete Ticket Lifecycle** - Create, view, edit, and manage tickets from open to closed
- **Status Management** - Three-state workflow: Open → In Progress → Closed
- **Priority** - 5-level priority ranking (1-Low to 5-Critical) for proper triage
- **Team Assignment** - Multi-assignee support with automatic email notifications
- **Search & Filtering** - Debounced search with status filtering and sorting
- **Infinite Scroll** - Smooth, performant pagination using Framer Motion
- **Authentication System** - JWT-based auth with role management (admin/developer)
- **Email Notifications** - Automatic notifications when tickets are assigned
